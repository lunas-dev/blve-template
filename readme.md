# blvedev/template

A template for creating a new Blve project.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en/)

### Installation

```bash
npm install -g tiged # Tool for copying this template
tiged blvedev/blve-template your-project-name
cd your-project-name
npm install
```

### Development

```bash
npm run dev
```
